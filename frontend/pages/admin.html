Pipeline

Core modules

  1. preprocessing.py Loads the PaySim CSV, drops nulls, one-hot encodes type, and keeps account IDs for graph work.
  2. graph_builder.py Builds a NetworkX directed graph where accounts are nodes and transaction amounts are edge weights.
  3. feature_engineering.py Calculates degree, PageRank, and clustering coefficient for transaction-level features.
  4. predict.py Updates the graph dynamically and returns fraud prediction plus probability.
