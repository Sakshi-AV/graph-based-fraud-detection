# Frontend Structure

- `pages/`: HTML screens served by Flask
- `static/css/main.css`: current dark theme stylesheet
- `static/js/app.js`: page bootstrapper
- `static/js/api.js`: shared fetch helpers
- `static/js/auth/`: user and admin login/logout flows
- `static/js/dashboard/`: admin and user dashboard rendering
- `static/js/monitor/`: transaction scanner logic
- `static/js/graph/`: graph data model and canvas renderer
- `static/js/data/`: local sample transactions and presets
